# recorded_bimanual_demo

## 用途

把一段 682 帧双臂关节示教压缩成按归一化阶段 `phase ∈ [0,1]` 输出 14 个关节角的轻量学生策略。
运行时只依赖 Python 标准库。

## 训练数据

- 来源：`userdata/znb/arm_follower_data.json`
- 双臂各 7 个关节
- 682 帧
- 原数据没有任务名称、物体位姿、RGB/深度、夹爪受力或成功标签

## 指标

- 模型大小：约 15 KB
- 轨迹 RMSE：约 0.0079 rad
- 最大绝对拟合误差：约 0.0712 rad

## 限制

- 这是开环轨迹模型，不会根据物体位置或对方机器人手臂位置修正动作。
- 原始轨迹用途未知，不能视为已学会交接。
- `validated=false`，真机运行时会被 Agent 拒绝。
- 不得通过删除安全检查直接用于持物或人机交互。

## 真机启用前

需要确认原始任务语义，完成关节限位与碰撞检查，并依次通过低速空载、软物体、轻负载和目标负载测试。
交接还需要接收方夹爪宽度/电流/腕力证据，才能允许交付方松手。
